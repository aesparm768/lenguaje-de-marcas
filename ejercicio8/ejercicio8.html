METALENGUAJES


SGML:
El Lenguaje de marcado generalizado para definir lenguajes de marcado declarativos.
XML:
Es un lenguaje de marcado que define un conjunto de reglas para la codificación de documentos.
RSS:
Es una tecnología que permite recibir la notificación de novedades de un servicio web o canal.
GML:
Es un archivo que contiene la información acerca de la georreferenciación o coordenadas de una parcela o una edificación.
XHTML:
Se utiliza para marcar contenido como texto, imágenes y enlaces en forma de hipervínculos para crear una cierta estructura que puede ser mostrada por los navegadores.
SVG:
Es un estándar web para definir gráficos basados en vectores en páginas web.
XUL:
Lenguaje basado en XML para la interfaz de usuario, desarrollado por Mozilla.
